Creating a Project

The very first step to get developing in Iron Coder is making a project.

Every project needs a name and at least one main board.

For this example, we will be creating a project my_project and using the

RP2040 as our main board.

For now, simply name the project, and then click start_development

If this is your first project, you will be asked to choose a directory to save

projects to. Afterwards, your project is ready!
